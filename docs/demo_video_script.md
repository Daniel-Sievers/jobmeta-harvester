# Demo-Video-/GIF-Skript

Dieses Skript ist für ein kurzes GIF oder ein Video von etwa 60 bis 90 Sekunden
gedacht.

## Empfehlung

- Nur Demo-Daten verwenden.
- Browserfenster auf 1280px Breite stellen.
- Persönliche Browserleisten, Lesezeichen und private Tabs ausblenden.
- Keine echten Bewerbungsnotizen zeigen.

## Ablauf für ein 60-Sekunden-GIF

### Szene 1: Dashboard öffnen

Bild:

- Dashboard mit Demo-Daten
- Metriken links
- Nächste Schritte

Text/Untertitel:

> Lokales Dashboard für strukturierte Jobsuche

### Szene 2: Filtern

Aktion:

- Mindestscore auf `70+`
- Ansicht auf `Analyse`

Text/Untertitel:

> Stellen werden nach Profil, Score und Skill-Lücken gefiltert

### Szene 3: Details öffnen

Aktion:

- `Details` bei einer passenden Stelle öffnen

Zeigen:

- Aufgaben
- Muss-Skills
- Lücken A/B/C
- Matching

Text/Untertitel:

> Jede Anzeige wird als Metadatensatz betrachtet

### Szene 4: Notiz speichern

Aktion:

- neutrale Notiz eintragen
- speichern
- Toast-Benachrichtigung zeigen

Text/Untertitel:

> Bewerbungsnotizen bleiben lokal in SQLite

### Szene 5: Browser-Clipper zeigen

Aktion:

- neutrale Demo-Stellenanzeige oder Beispielseite öffnen
- JobMeta-Clipper anklicken
- `Seite auslesen` klicken
- erkannte Felder kurz prüfen
- `An JobMeta senden` klicken

Text/Untertitel:

> Einzelne geöffnete Anzeigen können per Clipper übernommen werden

### Szene 6: Public-Export kurz erklären

Aktion:

- Keine normale Demo-Schaltfläche zeigen.
- Kurz auf die README-/CLI-Funktion verweisen.

Text/Untertitel:

> Public-Export entfernt private Daten automatisch und bleibt eine Entwicklerfunktion per Kommandozeile.

## Kurze gesprochene Version

> JobMeta Harvester ist ein lokales Python-Projekt für strukturierte Jobsuche.
> Das Tool importiert Stellenanzeigen aus APIs, CSV-Dateien oder über eine
> optionale Browser-Erweiterung, normalisiert sie als Metadaten, bewertet sie
> anhand eines Profils und macht Skill-Lücken sowie nächste Bewerbungsschritte
> sichtbar. Für GitHub gibt es einen Public-Export, der private Daten entfernt.

## Mögliche Tools

- OBS Studio für Videoaufnahme
- ScreenToGif für Windows-GIFs
- PowerPoint/Clipchamp für kurze Zuschnitte
- GitHub README für eingebettetes GIF
