Private Arbeitsversion

Die lokale Vollversion ist das Werkzeug für echte Bewerbungsdaten

Die Browser-PWA ist gut für Demo und schnelles Ausprobieren. Wenn du das Werkzeug ernsthaft privat nutzt, startest du die lokale Python-Version: gleiche Dashboard-Idee, aber mit SQLite-Datei auf deinem Rechner und ohne Cloud-Speicherung.

Zur Browser-PWA Demo ansehen Technische README

Startbefehl

Nach dem Entpacken der Public-Version im Projektordner:

pip install -e .
python -m jobmeta_harvester --dashboard

Danach im Browser öffnen:

http://127.0.0.1:8765

Warum lokal?

Echte Bewerbungsdaten, Notizen, CV-Profile und Statusfelder gehören nicht unnötig in eine öffentliche Demo. Die lokale Version speichert die Datenbank als Datei auf deinem Gerät und ist deshalb der richtige Modus für private Arbeit.

Was ist anders als in der PWA?

Die PWA läuft rein statisch im Browser. Die lokale Version hat ein Python-Backend und kann deshalb SQLite, robusteren CV-Import, echte erlaubte Quellen und den Public-Export per Kommandozeile besser unterstützen.
